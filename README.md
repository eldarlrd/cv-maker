## Install
```sh
bun i
```
## Develop
### Run Env
```sh
bun dev
```
### Prod Build
```sh
bun bundle
```
### Preview Prod Build
```sh
bun preview
```
### Format Code (ESLint + Stylelint + Prettier)
```sh
bun format
```
### Cut Orphans (Knip)
```sh
bun cut
```
## Deploy
```sh
bun deployer
```
